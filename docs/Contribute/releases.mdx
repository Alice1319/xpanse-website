# Releases

## Xpanse Website

We re-deploy the Xpanse website for every commit that is done to the `main` branch.

Continuous deployment is implemented based on GitHub actions as configured <Link name={'here'} url={'https://github.com/eclipse-xpanse/xpanse-website/blob/main/.github/workflows/xpanse-website-cd.yml'}/>.

## Xpanse Stack

All application necessary to run the Xpanse stack are released as docker images in the GitHub docker registry.
Java applications are additionally released as `jar` files which can be downloaded directly from the GitHub release notes.

Docker images can be found at <Link name={'GitHub Packages'} url={'https://github.com/orgs/eclipse-xpanse/packages'}/>.
