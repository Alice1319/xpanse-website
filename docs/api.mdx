# API

All functionalities of xpanse are externally exposed using REST API and follows OpenAPI specifications to document them.

It includes APIs for service vendors, end users, cloud service providers and system administrator.

## Swagger Hub

You can interact with the xpanse API using the <Link name={'REST API.'} url={'https://app.swaggerhub.com/apis/eclipse-xpanse-bot/XpanseRuntimeApi'}/>
